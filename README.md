# personal robot computer 个人机器人计算机

prc是以开发板作为主板，通常具有机械结构的个人计算机，可以通过web进行交互（模块是通用的，可以跑在正常pc）。


# 模块列表

## *pmc* 进程托管服务

该模块以C++写成，主要用于进程管理，可创建、托管子进程。

## *smn* 前端托管者

该模块可以像Apache一样托管前端页面，并且集成了代理功能，让其托管的页面能够访问到别的后端服务

## *jump* ssh跳板机

该模块让设备能够支持在web上访问ssh服务器

## *aibox* Agent智能体

该模块使得设备能够通过自然语言操作计算机。

## *Minervato* iot物联网

该模块部署在微控制器上面，让计算机能够以http方式与物理世界进行交互。

## Dashun 机器人硬件设计

机器人的图纸
